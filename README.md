# AI Code Review Assistant

## Features
- User Login & Signup
- Monaco Code Editor
- Static Code Analysis using ESLint
- AI Code Review
- Time & Space Complexity
- Code Smell Detection
- Function/API Documentation
- Review History
- Search & Filter Reviews

## Technologies
- Next.js
- Node.js
- Express.js
- PostgreSQL (Supabase)
- ESLint
- OpenAI API
- Monaco Editor

## Installation

Frontend
npm install
npm run dev

Backend
npm install
node index.js
